# React MovieDB 

React MovieDB is a movie search engine made with ReactJS and powered by [The Movie Database](https://www.themoviedb.org/ "TMDb") API.  

## Demo

https://www.tristan-lanoye.com/lab/react-moviedb/

- Webpack
- Babel
- Git
- Sass 
- CSS Grid
